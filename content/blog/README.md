# Статьи блога

Каждая статья — один `.md`-файл в этой папке. Имя файла = slug (URL: `/blog/<имя-файла>`).

Формат (frontmatter сверху, между `---`):

```
---
title: Заголовок статьи
description: Короткое описание для списка и SEO (1-2 фразы)
date: 2026-07-01
slug: pochemu-shkaf-polnyy
---

Текст статьи в Markdown. ## Подзаголовки, **жирный**, списки, [ссылки](/quiz).
```

Статьи пишет «Скилл статей» (`statejnik/`) — клади готовый `.md` сюда, и он сразу появится на `/blog`.
CTA на квиз и виджет-стилист добавляются автоматически. `README.md` в список не попадает.
