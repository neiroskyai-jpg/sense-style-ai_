# Hero photos for split-screen «До / После»

Положи сюда два файла **точно с такими именами**, иначе hero сломается:

- `01-do.jpg` — фото «До» (розовый трикотаж + джинсы на фоне Петроградки)
- `02-posle.jpg` — фото «После» (белый жилет + бежевые палаццо + красные туфли)

## Параметры

- Формат: JPG (или PNG, тогда в `index.html` поправить `.jpg` → `.png`)
- Соотношение: **3:4 портрет** (фото обрежется по `object-fit: cover`)
- Разрешение: от 800×1067, идеально 1200×1600 или больше — файл сожмётся в браузере
- Цветокоррекция не нужна — CSS сам приглушит «До» (лёгкий ч/б + desaturate) и оставит «После» в полном цвете
