# План: База коррекции фигуры (figure-correction)

**Дата:** 2026-06-09
**Функция:** Структурный справочник коррекции фигуры (5 типов + универсальные принципы + оптические иллюзии + вещи-корректоры) для AI-слоя. Второе звено движка после цветотипов.

## Зачем

`formula-diagnostic` и `look-generator` уже знают 5 типов фигуры, но правила коррекции в них короткие. Нужен глубокий справочник, как `colortypes/` и `style-library`, чтобы капсула собиралась с реальной коррекцией (баланс, пропорции, иллюзии), а не общими словами. Источник — «Модуль 5. Типы фигур и способы коррекции» + конспекты «Силуэты», «Иллюзии», «Универсальные луки» (raw-inbox, есть текстовый слой).

## Фазы

### [x] Фаза 1. Извлечение
- Извлечён текст Модуля 5 (44k символов) + Силуэты + Иллюзии + Универсальные луки.
- Зафиксированы: 5 типов (особенности/задача/подходит/избегать/лучший предмет), универсальные принципы, иллюзии, вещи-корректоры.

### [x] Фаза 2. Оформление справочника
- `architecture/reference/figure-correction/figure-correction.md` v1.0 + README.
