# План: глубокий слой диагностики внутри Карты (тело+возражения → личность)

**Цель:** добавить в платную Карту стиля глубину из каноничной анкеты курса: тело+возражения
(сейчас), потом личность Big Five. НЕ раздувать бесплатный квиз — глубокая диагностика идёт в
форме сборки Карты. Источник: [[client-intake-anketa]].

**Решения фаундера:** личность — внутрь Карты (одна покупка, больше глубины); начинаем с тело+возражения.

## Фаза 1. Тело + возражения [x]
4 поля из анкеты в `CARD_BUILD_FORM` (все опциональные, body-positive):
- [x] **Достоинство** (adv) · **Баланс** (balance) · **Табу** (taboo) · **Чьё мнение** (audience) — поля в форме.
- [x] `/card/build`: `_save_deep_intake` → в профиль (`diagnosis.deep_intake`) до старта воркера.
- [x] `build_style_card`: табу → `stop_list` (без дублей); достоинство/баланс → `gen_req` (emphasize/balance);
  достоинство показываем в «Твоя основа» (Подчёркиваем).
- [x] Чат-стилист: deep_intake в `_profile_block` (подчёркивает достоинство, не предлагает табу, мнение→к себе).
- Проверено: синтаксис, поля формы, парсинг табу в стоп-лист, 43 теста.

## Фаза 2. Личность Big Five [x]
Решение фаундера: НЕ архетипы (ненаучно + риск «шаблона» по исследованию рынка) — **персональный портрет**.
+ 2 вопроса из анкеты, которые меняют выдачу (круг жизни, бюджет).
- [x] Тест: 10 утверждений (`BIG5_QUESTIONS`, 2 на черту O/C/E/A/S), шкала 1–5, S реверсивна; `_score_big5` → high/mid/low.
- [x] `generate_personality_portrait` (flash): уровни → ЖИВОЙ портрет + 3 вывода для стиля, БЕЗ ярлыков/процентов/архетипов, tone of voice. Проверено вживую.
- [x] Форма Карты: секция «Пара вопросов о тебе» (10 шкал) + «Круг жизни» (% работа/дом/свободное) + бюджет.
- [x] `build_style_card`: бюджет → price_segment (капсула+шоп-лист); круг жизни → gen_req; портрет → раздел «Твоя натура и стиль».
- [x] Портрет сохраняется в профиль (`personality_portrait`) → чат-стилист персонализируется.
- Проверено: форма, подсчёт, портрет, рендер, 43 теста.

## Итог
Глубокий слой Карты реализован целиком: тело+возражения + личность (Big Five-портрет) + круг жизни + бюджет.
Всё опционально, в платной Карте, свободный квиз не тронут. Источник — каноничная анкета курса [[client-intake-anketa]].
