# База актуальных образов из магазинов по партнёрской программе

**Идея:** перестать показывать только AI-фантазию и опереть рекомендации на **реальные покупаемые вещи** из магазинов-партнёров. Образ = конкретные товары со ссылками.

**Зачем (смена опоры продукта):**
- **Монетизация** — партнёрские ссылки = доход с каждой рекомендации.
- **Достоверность** — ИИ не выдумывает несуществующие вещи; привязка к реальному сезону/ассортименту.
- **Доверие** — «вот жакет за N ₽, вот ссылка» = работа стилиста, а не абстракция.

**Связь с тем, что уже есть:**
- `generate_shopping_list` + `marketplace_links` (WB/Lamoda/Ozon) — зачаток шоп-слоя.
- **RAG** (в работе) — тот же retrieval, но извлекаем не «правила метода», а «реальные вещи под профиль фигура/формула/сезон/цвет». Один механизм.
- **Генерация** — реальную вещь можно подать референсом в Nano Banana Pro (мультиреференс) → клиентка видит себя в **покупаемой** вещи (идеал: персонально + по ссылке).

## Фаза 1 — Узкий срез (proof) `[ ]`
- 1-2 партнёра (фид или ручная выгрузка).
- 1 формула × 1 фигура (например, Натуральный × прямоугольник) — собрать ~20-30 вещей.
- Теги: категория, фигура, стиль/формула, сезон, цвет/палитра, цена, ссылка, фото.
- Показать капсулу из РЕАЛЬНЫХ товаров со ссылками на экране результата.

## Фаза 2 — Подбор под профиль `[ ]`
- Логика матчинга: профиль диагностики → фильтр/ранжирование каталога (через RAG-индекс).
- Учёт stop_list (табу-цвета/силуэты), бюджета, фигуры.

## Фаза 3 — Каталог как референс генерации `[ ]`
- Фото товара + фото клиентки → рендер «она в этой конкретной вещи».
- Образ кликабелен → корзина партнёра.

## Фаза 4 — Масштаб `[ ]`
- Больше партнёров, автообновление фида, ротация по сезону, контроль наличия/цены.

## Источник каталога (уточнено 2026-06-29)
- **Lamoda → через CPA-сеть Admitad** (не через API продавца Lamoda — тот для селлеров FBO, нам не подходит).
- Admitad даёт **XML-фид товаров** (название, цена, бренд, категория, цвет, фото, deeplink) + партнёрские deeplink'и с трекингом.
- **Комиссия:** 12% с оплаченных заказов (3% через купоны, 9% через кешбэк). Cookie 30 дней. Выплаты от 1000 ₽, 2 раза/мес.
- Путь: регистрация в Admitad → подключить оффер Lamoda → забрать XML-фид → deeplink на каждую вещь.
- Так же подключаются другие магазины в Admitad (один фид-механизм на всех).
