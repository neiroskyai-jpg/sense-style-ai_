# План: База колориста — 12 цветотипов (colortypes)

**Дата:** 2026-06-09
**Функция:** Структурный справочник 12 цветотипов (палитры, нейтрали, металлы, табу-цвета) для AI-слоя. Недостающее звено перед генерацией капсулы: определённый цветотип → жёсткая палитра в `visual_formula`, которую потребляют `look-generator` и `formula-diagnostic`.

## Зачем

Проверка `look-generator` на кейсе Марии показала: палитра в формуле сейчас берётся «из воздуха», цветотип определяется нестрого — и капсула одевает клиентку не в её цвета. Нужен строгий колор-пасс по системе 12 цветотипов с табу-цветами, которые нельзя перекрывать. Источник данных — палитры курса «Алгоритмы имиджа» (картиночные PDF в `architecture/reference/raw-inbox/`: Весна/Лето/Осень/Зима × контрастная/натуральная/светлая).

## Фазы

### [x] Фаза 1. Извлечение палитр
- Отрендерены 12 палитр (PyMuPDF) в PNG, прочитаны swatch-сетки.
- Зафиксированы по каждому типу: характер (тепло/холод, чистота, глубина, контраст), базовые, акцентные, нейтрали (вместо чёрного/белого).

### [x] Фаза 2. Металлы и табу
- По каждому типу проставлены металлы и табу-цвета.
- Достроены по 12-сезонной логике (грид показывает «свои» цвета, не запретные), сверены с подтверждённым эталоном Натурального Лета. **Помечены в справочнике как требующие валидации Ксенией.**

### [x] Фаза 3. Оформление справочника
- Создан `architecture/reference/colortypes/12-cvetotipov.md` v1.0 + README папки + README `architecture/reference/`.

### [x] Фаза 4. Интеграция
- [x] В `look-generator`: добавлено жёсткое правило — палитра/табу приходят из цветотипа (ссылка на справочник), табу-цвета цветотипа перекрывать нельзя.
- [x] В `formula-diagnostic`: в ШАГ 7 добавлен шаг определения цветотипа (ссылка на справочник, табу всегда в stop_list) + поле `colortype` (12 ключей) в OUTPUT SCHEMA.

### [ ] Фаза 5. Валидация Ксенией
- Сверить палитры и табу с её колор-анализом (эталон — Натуральная зима Марии).

## Итог

Реализован почти целиком (фазы 1-3 готовы, фаза 4 на 50%). Готов справочник `12-cvetotipov.md` v1.0 и встроен в `look-generator`. Осталось: шаг цветотипа в `formula-diagnostic` (фаза 4) и валидация Ксенией — особенно металлы и табу-цвета (фаза 5).
