# План: База прототипов (prototype-library)

**Дата:** 2026-06-09
**Функция:** Справочная база подбора прототипа (визуального вектора) для AI-слоя. Подключается к `formula-diagnostic` ШАГ 5.

## Зачем

Шаг «прототип» уже есть в `formula-diagnostic` (ШАГ 5, двойная проверка + фильтр по цветотипу), но у него, в отличие от цветотипа/фигуры/принтов, **не было своей справочной базы**. ИИ выбирал прототип «из головы» — главный риск по предупреждению директора по стилю («ИИ выдаёт вероятные, но случайные ответы, проверяй»). Нужна выверенная библиотека, из которой ИИ берёт прототип, а не генерирует.
