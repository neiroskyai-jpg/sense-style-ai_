# План: психотип определяет подстиль (шаг 4 метода) — платная глубина Карты

**Задача:** сделать психотип (Big Five) ДВИЖКОМ подстиля, а не декоративной секцией.
Метод, шаг 4: *подстиль определяется возрастом, профессией и психологическим портретом
(закрытая/открытая, демонстрирующая/прячущая)*. Сейчас `diagnose()` выбирает подстиль в момент
квиза без психотипа, а Big Five из формы Карты уходит только в `generate_personality_portrait`
(портрет сбоку, на подстиль не влияет). Из-за этого платная Карта уходит вширь, а не вглубь —
и её заголовок «Инструкция к твоему психотипу» не подкреплён.

**Решение фаундера (2026-07-02):** делаем движок ПЕРВЫМ, до обновления копии сайта; тарифная
лестница остаётся 0 → 3900 → 14900 → 35000 → подписка (среднюю ступень возвращаем).

## Что меняем
Психотип превращает базовый стиль (4) → подстиль (25). Бесплатно = базовый стиль; платно =
подстиль + «почему именно этот подстиль из твоей натуры». Возраста/профессии в форме Карты нет —
драйверы: `big5` (психотип), `want_styles` (визуальный выбор), `lifecircle`.
