# Принцип 2 — Системность

> Источник: [.business/company/values.md](../../.business/company/values.md).

## Суть

Гардероб = база данных и алгоритм, а не эмоциональный хаос. AI в продукте — строгий инструмент для построения системы, а не развлекательная фича «миксующая шмотки».

## Как это работает в Premium

- 7-шаговый алгоритм диагностики (см. [[../06-architecture/method-7-steps]]).
- 4 семантических поля → 4 базовых стиля → 25 подстилей → 5 типов фигуры.
- Каждый блок Style Book проверяется по чек-листу контроля качества (10 пунктов).
- Капсула = структура (база + акценты + аксессуары), а не «коллекция образов».
