## 1. Концепция продукта

**Sense Style** — стилист на основе психологии моды. Применяет авторскую методологию Ксении Колупаевой (школа «Алгоритмы имиджа» + современная психология моды Carolyn Mair, Лондонский колледж моды, 2025) и AI-инструменты для генерации персональных образов.

### Главное обещание клиентке

Узнать свою **Формулу стиля** и увидеть себя в идеальных образах под фигуру и сценарии жизни.

### Главное отличие от конкурентов

- **AI-стилисты-конкуренты** работают с цветотипом или собирают случайные комбинации из готовых вещей. Без методологии, без психологии, без живого стилиста.
- **Sense Style** работает с идентичностью через концепцию **разрыва идентичности** (расхождение между тем, как клиентку считывают, и тем, как она хочет, чтобы её считывали). На стыке профессиональной школы стиля + психологии моды + AI.

Ни один продукт на рынке не соединяет: **методологию профессионального стилиста + современную психологию моды + AI-генерацию образов под фигуру + русский язык + premium-позиционирование**. Это окно занимает Sense Style.

## 2. Целевая аудитория

### Основной сегмент
- Женщины **30–50 лет**
- Доход средний+ и выше
- В точке профессионального или личностного перехода
- Готовы платить за **результат**, не за «контент»
- Ценят системность и профессиональное обоснование

### Боли клиенток
- «Шкаф полный — нечего надеть»
- «Не понимаю, как выглядеть на свой возраст и статус»
- «Хочу транслировать определённое впечатление, но не знаю как»
- «Покупаю одежду импульсивно, потом не ношу»
- «Не доверяю онлайн-стилистам — все одинаково собирают случайные образы»

### Поведение
- Активно используют Pinterest, Instagram-блогеров, фэшн-журналы
- Покупают в Toteme, COS, 12Storeez, Massimo Dutti, Anna Pekun, премиум сегменте
- Готовы попробовать AI-инструмент, если он опирается на профессиональную методологию

## 3. Функционал — приоритеты

### P0 — без этого нет продукта

#### 3.1. Квиз-диагностика (12 вопросов, 3 минуты)
- Контекст жизни и роли (распределение времени, дресс-код)
- Имидж-диагностика (как тебя считывают / как хочешь)
- 3 главные характеристики желаемого впечатления
- Тональные характеристики внешности
- Параметры (для типа фигуры)
- Тренды-ориентиры и ценовой сегмент

Результат: **разрыв идентичности (%) + Формула стиля + 1 образ-превью**.

Алгоритм определения Формулы стиля — в `architecture/sense-style-method.md` раздел 7.

#### 3.2. Формула стиля
Персональная комбинация из 2-3 направлений по авторской методологии Sense Style. Например: «Минимализм × Power Woman × Drama-акцент».

Не архетип («ты — Драматик»), а **гибкая формула** под конкретную жизнь и роли.

#### 3.3. Загрузка фото в полный рост — ВНУТРИ квиза (обновлено 2026-05-19)
Клиентка прикладывает свой снимок на **предпоследнем шаге квиза**, перед финальным экраном. AI получает: фото + Формула стиля + тип фигуры + тональные характеристики.

**Изменение от 2026-05-19:** загрузка фото перенесена с отдельной страницы `/upload` (через email) внутрь квиза для WOW-эффекта в момент максимального вовлечения. Подробнее — см. `plans/2026-05-19-ai-photo-generation.md`.

#### 3.4. Генерация AI-образов
- **Финальный экран квиза (бесплатно):** 2 фото клиентки в её формуле — главное направление + нота. Это WOW-якорь для покупки «Карты стиля».
- В тарифе «Стилевая карта»: 6 фотореалистичных образов под сценарии (деловая встреча, презентация, свидание, выходные, корпоратив, путешествие)
- В тарифе «Книга стиля»: дополнительные капсулы под жизненные роли
- В тарифе «Перерождение образа»: 12 сценарных образов + план перехода + видеоразбор

Каждый образ — с лицом и фигурой клиентки, под её Формулу стиля.

#### 3.5. Платёжная система
- **3 разовых тарифа**: 2 900 / 14 900 / 15 000 ₽
- **1 подписка** (P2): 3 900 ₽/мес
- **ЮKassa Checkout** — единственный эквайринг на этапе MVP (ИП РФ / самозанятая)
- Webhook для статуса оплаты

#### 3.6. Email-уведомления
- Через Resend
- На каждом этапе флоу

### P1 — первый месяц после MVP
- Лендинг с GA4 + Meta Pixel
- Личный кабинет с историей образов
- PDF-экспорт стайл-бука

### P2 — после 30 продаж

- Капсулы по ролям

- Сезонные обновления

## 4. Технический стек

### Frontend
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (анимации)

### Hosting
- Vercel (free tier на старте)
- Автодеплой из GitHub

### База данных
- Supabase PostgreSQL
- Supabase Auth (magic link)
- Supabase Storage (фото клиентки, AI-образы, PDF)
- Supabase Edge Functions (триггер AI-генерации)

### AI-слой
- **Claude Vision API** — анализ фото клиентки (тональные характеристики, тип фигуры, лицо)
- **`claude-sonnet-4-6`** — генерация Формулы стиля и описаний образов на основе методологии
- **Image generation API** (Nano Banana / Midjourney / Stable Diffusion via Replicate) — генерация фотореалистичных образов с лицом клиентки

Системный промпт для генерации опирается на `architecture/sense-style-method.md` целиком.

### Платежи
**Юр.форма фаундера: ИП РФ или самозанятая РФ.**
- **ЮKassa** — единственный эквайринг. Прямое подключение к ИП/самозанятой, рублёвые расчёты, карты + СБП + Tinkoff Pay.
- **ЮKassa Recurring** — для подписки (P2).
- **Stripe** — отложен до международной экспансии. Не работает напрямую с российским ИП.

### Email и аналитика
- Resend (транзакционные)
- Google Analytics 4 + GTM
- Meta Pixel
- OpenWeatherMap API (для подписки P2)

## 6. Тарифная сетка

5 уровней — от бесплатной диагностики до работы со стилистом.

### Тариф 1. Диагностика идентичности — 0 ₽
**Вход в продукт.** Бесплатно навсегда.
- Квиз-диагностика (12 вопросов, 3 минуты)
- Расчёт разрыва идентичности (%)
- Формула стиля (комбинация из 2-3 направлений)
- 1 образ-превью

### Тариф 2. Стилевая карта — 2 900 ₽
**Чаще всего выбирают.** Разовый платёж.
- Полная Формула стиля с описанием
- 6 готовых образов под фигуру (фотореалистичные AI с лицом клиентки)
- Сценарии жизни (деловая встреча, свидание, выходные и т.д.)
- Список покупок с конкретными брендами

### Тариф 3. Книга стиля — 14 900 ₽
**Главный продукт.** Разовый платёж.
- Капсулы по ролям (4 ролевые капсулы)
- Анализ текущего гардероба
- Готовая палитра под тональные характеристики
- Мудборды на сезон (весна-лето + осень-зима)
- Рекомендации по макияжу и причёске

### Тариф 4. Перерождение образа — 15 000 ₽
**Премиум.** Разово, мест ограничено.
- Глубокий разбор стилевого разрыва
- 12 сценарных образов

### Тариф 5. Sense Style каждый день — 3 900 ₽/мес
**Подписка (P2).** Скоро.
- Образ дня под погоду
- Капсулы по ролям
- Трекер разрыва идентичности
- Сезонные обновления

- Тариф 1 (бесплатно) — 100% посетителей

## 7. Ключевые пользовательские сценарии

### Сценарий 1. Бесплатная диагностика
1. Клиентка попадает на лендинг через Reels / Pinterest / Telegram
2. Жмёт «Пройти диагностику бесплатно»
3. Проходит квиз 12 вопросов (3 минуты)
4. На странице результата получает: разрыв идентичности %, Формулу стиля, 1 образ-превью
5. CTA: «Получить полный набор образов» → переход к выбору тарифа

### Сценарий 2. Покупка тарифа «Стилевая карта»
1. После результата квиза клиентка выбирает «Стилевая карта 2 900 ₽»
2. ЮKassa Checkout → оплата
3. Webhook обновляет статус → «paid»
4. Email от Resend: ссылка на загрузку фото
5. Клиентка загружает фото в полный рост
6. Edge Function запускает Claude Vision анализ + генерацию 6 образов через image generation API
7. Email: ссылка на готовые образы (PDF + онлайн-галерея)
8. Время от оплаты до результата: < 30 минут

### Сценарий 3. «Книга стиля» (главный продукт)
1. Клиентка выбирает тариф «Книга стиля 14 900 ₽»
2. ЮKassa Checkout → оплата
3. Email: анкета с дополнительными вопросами (роли, дресс-код по сферам, имеющийся гардероб)
4. Клиентка заполняет, загружает фото своих вещей
5. AI генерирует:
   - 4 ролевые капсулы
   - Анализ имеющегося гардероба (что работает, что отпустить)
   - Палитра + мудборды
   - Список точечных покупок
6. Стилист (Ксения) делает финальную проверку через чек-лист контроля качества (методология раздел 10)
7. Email: PDF-стайл-бук
8. Время: 2-3 дня

## 8. Структура базы данных

### `users`
- `id` (uuid), `email`, `subscription_status`, `created_at`

### `gap_profiles`
- `user_id`
- `gap_percentage` (разрыв идентичности 0-100)
- `style_formula` (text — например «Минимализм × Power Woman × Drama-акцент»)
- `now_traits` (array — как считывают сейчас)
- `want_traits` (array — 3 главные)
- `tonal` (json — подтон, глубина, контраст)
- `figure_type` (enum — прямоугольник / песочные часы / груша / перевёрнутый треугольник / яблоко)
- `roles` (array — work, personal, events)
- `quiz_answers` (json — все ответы квиза)

### `orders`
- `id`, `user_id`
- `tier` (enum: free / style_card / style_book / transition / subscription)
- `amount`, `provider` (yukassa), `provider_payment_id`
- `status` (pending / paid / failed)
- `created_at`

### `looks`
- `id`, `user_id`
- `scenario` (deal_meeting, presentation, date, weekend, corporate, travel)
- `image_url` (Supabase Storage)
- `description` (AI-объяснение через Формулу стиля)
- `created_at`

### `wardrobe_items` (тариф «Книга стиля»)
- `id`, `user_id`
- `photo_url`
- `category`, `color`, `style`
- `verdict` (works / let_go / accent)
- `explanation`

### `capsules` (тариф «Книга стиля»)
- `id`, `user_id`
- `role` (work / personal / events)
- `items` (json array)
- `pdf_url`

## 12. Открытые вопросы

1. **Согласие Микуловой** на использование её AI-образов в кейсе «из практики». Устного «разрешает» недостаточно для коммерческой публикации.
2. **Юридический статус упоминания школы «Алгоритмы имиджа»** — согласовать с ними публичное упоминание / показ диплома.
3. **Договор-оферта + политика конфиденциальности** — обязательны до подключения ЮKassa.
4. **Image generation API** — выбор: Replicate (Nano Banana / SDXL) vs Midjourney API vs собственный fine-tune. Влияет на качество и стоимость.
5. **Промпт-инжиниринг** для генерации образов под Формулу стиля — требует серии тестов на 10-15 кейсах перед запуском.
6. **Бренд-активы** — логотип, фавикон, иконки методологии (нужны до публичного запуска).
