# architecture/reference/image-generation/

База генерации образов — knowledge base для AI-слоя. Как собирать фотореалистичный `image_generation_prompt` для preview-якоря квиза.

| Файл | Что внутри | Статус |
|---|---|---|
| `image-generation.md` | Структура промпта (6 блоков), фотореалистичный тулкит (камеры/плёнки/стиль), маппинг Формула→фотостиль, сохранение идентичности | ✅ v1.0, требует валидации |

## Назначение

`formula-diagnostic` заполняет `preview_look.image_generation_prompt`: клиентка (лицо/фигура с фото) в её Формуле стиля, в фотореалистичном качестве. Палитра без табу-цветов, силуэт по фигуре. 2 бесплатных образа = WOW-якорь покупки.

## Источник

«Структура промпта», «Чек-лист по фотореалистичным генерациям» («Алгоритмы имиджа»). Инструмент курса — Midjourney/SYNTX; принципы тул-агностичны.
