# architecture/reference/prototypes/

База подбора прототипа — knowledge base для AI-слоя. Прототип = визуальный вектор желаемого впечатления (публичная фигура), не шаблон.

| Файл | Что внутри | Статус |
|---|---|---|
| `prototype-library.md` | Метод подбора (двойная проверка + фильтр по цветотипу) + растущая библиотека выверенных прототипов | ✅ v1.0; библиотека наполняется Ксенией |

## Назначение

`formula-diagnostic` ШАГ 5 берёт прототип отсюда по полю впечатления + фигуре + контрасту, затем фильтрует образы прототипа под цветотип клиентки (`../colortypes/`). ИИ не выдумывает прототип «из головы» — только из библиотеки; иначе помечает кандидата «черновик, подтвердить Ксенией».

## Источник

7-шаговый метод (`../../sense-style-method.md`) + дипломная методика Ксении, подтверждённая директором по стилю школы «Алгоритмы имиджа».
