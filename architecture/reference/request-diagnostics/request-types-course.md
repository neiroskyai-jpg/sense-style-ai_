# Типы запросов клиентки и переформулирование (канон курса image66)

**Тип:** knowledge base для AI-слоя. Питает `request-diagnostics`, **чат-стилиста** (`core/chat.py`)
и диагностику. Цель: ловить НАСТОЯЩИЙ (аутентичный) запрос и мягко переводить неконструктивный
в конструктивный — «говорить с клиенткой на одном языке». Источник: конспект «Типы запросов у клиентов».

## Типы вопросов (как стилист спрашивает)
1. **Информативные** — получить новую информацию (что, как, где, сколько).
2. **На проверку гипотезы** — уточнить точность понятого («правильно ли я понял, что…»).
3. **На введение новой информации** — ненавязчиво познакомить («а вы знали, что…»).

## Запросы: конструктивные ↔ неконструктивные

Конструктивный запрос = **про себя, свои чувства/ощущения, достижимый, без обобщений и без третьих лиц.**
Неконструктивные бывают трёх видов — их надо РАСПОЗНАТЬ и переформулировать:

### 1. Нереалистичные (обобщения: всегда / никогда / на 100%)
- «Хочу выглядеть на все 100%» → **«в большинстве ситуаций выглядеть стильно»**
- «Никогда больше не покупать чёрного» → **«стараться не покупать чёрные вещи»**
- «Всегда нравиться мужу» → **«часто чувствовать, что я нравлюсь мужу»**
- Алгоритм: 1) переформулировать в утверждение, 2) признать, что «всегда/100%» невозможно, 3) собрать конструктивный запрос.

### 2. Неопределённые (абстрактные, «заумные», описание ситуаций)
- «У меня всё плохо, ничего не радует» → **«хочу порадовать себя одеждой, нравиться себе»**
- «Я позитивный человек» → **«хочу, чтобы одежда отражала мой позитив»**
- «Хочу, чтобы всё получалось» → **«хочу, чтобы одежда отражала мою индивидуальность»**
- Алгоритм: 1) спросить «чем вам может помочь стилист?», 2) собрать конструктивный запрос.

### 3. Манипулятивные (про третьих лиц / других людей)
- «Хочу, чтобы мама мной гордилась» → **«хочу гордиться собой»**
- «Хочу, чтобы муж не смотрел на других» → **«хочу чувствовать уверенность в своей красоте»**
- «Хочу, чтобы коллеги не сомневались в моей компетентности» → **«хочу чувствовать себя компетентной»**
- Алгоритм: переформулировать запрос относительно СЕБЯ, своих чувств и эмоций.

## Как переформулировать неконструктивный запрос (5 шагов)
1. Распознать, что запрос неконструктивный (научиться их видеть).
2. Дать клиентке обратную связь, почему с такой формулировкой не работаем.
3. Спросить: «Что вы можете сделать, каким быть, чтобы достичь желаемого результата?»
4. Сформулировать конструктивный запрос (про себя, достижимый).
5. Порадоваться вместе.

## Как подключается
- **Чат-стилист** (`core/chat.py`): распознаёт неконструктивный запрос и МЯГКО переводит в
  конструктивный (про себя, достижимый), на «ты», в tone of voice — это и есть «один язык».
- **Диагностика**: want_traits лучше принимать уже в конструктивной форме (про себя, без «всегда/никогда»).
