# Анкета клиента + метод Identity Gap (канон курса «Алгоритмы имиджа»)

**Тип:** knowledge base для AI-слоя. Чертёж интервью/диагностики. Питает квиз, `formula-diagnostic`,
**чат-стилиста**. Источник: «Анкета для работы с клиентом» (часть 1 — параметры, часть 2 — метод разрыва).
Это каноничная реализация Self-Discrepancy: впечатление СЕЙЧАС → ЖЕЛАЕМОЕ → разрыв → цель образа.

## Часть 1. Базовые данные (21 пункт)
ФИО; город/страна; **параметры** (обхваты груди/плеч/талии/бёдер); рост, размер обуви, размер одежды
верх/низ; возраст, профессия; **хобби/образ жизни**; **главное достоинство внешности (Q7)**;
**что хотела бы скорректировать (Q8)**; любимые цвета; любимые бренды; **стильные табу (Q11)**
(«не ношу красный / мини / каблук >5 см»); макияж/укладки ежедневно?; **3 прилагательных — желаемое
впечатление (Q13)**; носит ли аксессуары + сложности с ними; **бюджет на шоппинг (Q16)**;
**чей стиль вдохновляет (Q17)**; соцсети; **чьё мнение учитываем — мама/дети/муж (Q19)**;
особые пожелания; принципиальна ли натуральная кожа.

> Тело/возражения для диагностики: Q7 (достоинство — что подчеркнуть), Q8 (коррекция — body-positive),
> Q11 (табу — стоп-лист), Q19 (третьи лица — связь с манипулятивным запросом, см. [[request-types-course]]).

## Часть 2. Метод разрыва (Identity Gap) — ядро
1. **Круг жизни:** где и сколько времени в неделю (% дом/семья · работа · свободное время). → под это капсулы.
2. **Для чего нужна одежда** (вывод из круга): по каждой зоне жизни — какой запрос (пример: «для работы без
   дресс-кода, но не джинсы»; «дома тепло и женственно»).
3. **Целевая аудитория образа:** кто видит (коллектив, муж, дети) и чего от образа ждёт окружение.
4. **Впечатление СЕЙЧАС:** отметить 10+ характеристик из списка (реальное Я).
5. **Впечатление ЖЕЛАЕМОЕ:** отметить 10+ характеристик (идеальное Я). Часть может совпадать с п.4.
6. **РАЗРЫВ:** в чём реальный имидж расходится с желаемым; чего не хватает в гардеробе и поведении.
   Пример: «всё чёрное/серое, не хватает цвета; кажусь закрытой — хочу быть открытой».
7. **ВЫВОД — цель образа:** пример: «гардероб под 2 цели: на работе авторитетная/компетентная, дома
   романтичная/лёгкая; добавить цвет, перестать покупать только чёрное».

## Список характеристик впечатления (каноничный лексикон, 4 группы ≈ 4 семантических поля)
- **Авторитет/статус:** организованная, собранная, сдержанная, благородная, притязательная, дорогая,
  превосходная, надёжная, авторитарная, авторитетная, компетентная, честолюбивая, уверенная, элегантная,
  солидная, серьёзная, властная, амбициозная.
- **Открытость/натуральность:** деловитая, независимая, простая, солидарная, доверительная, открытая,
  коммуникабельная, беззаботная, непритязательная, толерантная, непосредственная, располагающая,
  экологичная, весёлая, любознательная.
- **Женственность/романтика:** привлекательная, женственная, романтическая, очаровательная,
  обольстительная, легкомысленная, желанная, роковая, эффектная, импозантная, несравненная, шикарная,
  чувствительная, нежная, ранимая, загадочная.
- **Креатив/драма:** смелая, неординарная, эксцентричная, провокационная, умная, новая, незаурядная,
  эпатирующая, одинокая, странная, особенная, прячущая, демонстрирующая, отстранённая, вдумчивая,
  замкнутая, ищущая.

> Это валидирует наши 4 семантических поля и даёт каноничный словарь для now_traits/want_traits.
> Разрыв между «сейчас» и «желаемым» = Identity Gap %, который мы и замеряем.

## Как подключается
- Квиз/диагностика: now_traits/want_traits — из этого списка; круг жизни → веса сценариев капсулы;
  Q7/Q8/Q11 → тело и стоп-лист; Q19 → деликатно снять «про третьих лиц».
- `formula-diagnostic`: цель образа (п.7) = вектор Формулы; разрыв (п.6) = что закрываем.
