# Универсальные луки и капсула по типам фигур (канон курса «Алгоритмы имиджа»)

**Тип:** knowledge base для AI-слоя. Питает `look-generator` (капсула/образы), `garment-check`,
`vision-analyzer` (пропорции). Источник: курсовые «Универсальные луки по типам фигур», «Снятие
плоскостных и объёмных характеристик». Цель: рекомендации носибельные, под фигуру, как у стилиста.

## Вещи-корректоры — комплиментарны ЛЮБОЙ фигуре (даже plus size и кругу)
Полуприлегающий мягкий неструктурный крой, ткань средней плотности. Безопасная **базовая капсула**:
1. Юбка на запах
2. Классический тренч (а-ля Burberry)
3. Джинсы MOM
4. Костюм пижамного кроя
5. Худи / свитшоты
6. Платье-рубашка и платье на запах (иллюзия Мюллера-Лайера — двойной V-вырез; лучше на округлых формах)
7. Классическая мужская рубашка
8. Мягкий костюм
> Это дефолтное ядро капсулы, если фигура определена неуверенно: почти беспроигрышно.

## Безошибочные комбинации по фигуре

**ПЕСОЧНЫЕ ЧАСЫ** (важно частично оголять тело, не допускать глухого закрытия; не оверсайз верх+низ одновременно):
1. Платье-футляр; 2. Трикотажное платье, обрамляющее фигуру; 3. Кроп-топ/бра + деним (mom's/straight) + каблук;
4. Юбка-мини + объёмный свитер/жакет + массивный низ (челси, грубые ботинки); 5. Костюм мягкой формы, слим/приталенный.
Брюки — с заужением книзу. При закрытом верхе — акцент на длинных ногах.

**PLUS SIZE «песочные часы» / «гитара»** (животик + резкий перепад талия-бедро, высокое крутое бедро; высокая талия, без низкой посадки; избегать прилегания и тонких тканей на животе):
1. Платье на запах; 2. Формодержащий жакет-корректор слим + джинсы зауженные книзу; 3. Блузы/юбки на запах, высокая талия;
4. Плотная рубашка хлопок/деним + крупный пояс (не «ниточка») + брюки-дудочки; 5. Пижамный костюм.

**ВЕРХНИЙ ТИП** (массивный верх — принцип «укрытия»: перекрываем объёмной курткой/жакетом ниже середины бедра, делаем «коробочку», подчёркиваем длинные ноги):
1. Слим легинсы / клёш от колена + оверсайз жакет + массивная обувь; 2. Скинни/straight/MOM + оверсайз джинсовка ниже середины бедра;
3. Объёмные худи/свитшоты + деним, свитшот можно повязать на бёдра (уравновешивает); 4. Платья-рубашки/балахоны/с пышным низом мини + ботфорты;
5. Платье-жакет (соло или со слим юбкой).

**НИЖНИЙ ТИП** (объём в бёдрах/галифе — «воздух» в зоне бёдер, без облегания, минимум декора/складок на бёдрах; монохром вытягивает):
1. Платье плотного трикотажа А-силуэт миди/мидакси; 2. Фактурный свитер + струящиеся палаццо/юбка в бельевом стиле (свитер может перекрывать галифе);
3. Плотные юбки А-силуэт + ремень на талии + монохром; 4. Костюм-двойка/тройка с мягким жакетом-корректором;
5. Однобортный удлинённый жакет с подплечниками, мягкий крой (без лишней геометрии). Каблук/нюд-обувь для лёгкости.

**ПРЯМОЙ ТИП** (талия формируется искусственно — объём сверху и снизу одновременно, либо ремень + складки/драпировки):
1. Рубашка-разлетайка / V-вырез + джинсы wide-leg/slouch; 2. Комбинезон с ремнём + каблук;
3. Платья baby-doll, на запах, трапеции (отрезная талия), со сборкой на талии, асимметрия мини; 4. Слим жакеты и брючные костюмы.

## Простые комбинаторные решения (демо капсульности)
- **Худи** легко сочетается: длинная атласная юбка + каблук/кроссовки; широкие брюки; джинсы wide-leg + кроп/бомбер.
- **Джинсы** легко сочетаются: базовый топ/футболка/рубашка/жакет + корсет/лодочки/очки — деловой и расслабленный образ из одной вещи.

## Чек-лист гардероба (весна 2026)
Тренч/шерстяное пальто (натуральная шерсть); мягкий пиджак кэжуал-классика; косуха/замшевый жакет;
3-4 разноплановых низа (белые джинсы, шёлковая юбка мидакси в бельевом стиле, палаццо);
5-6 верхов (свитер-зипка, рубашка, кашемировая водолазка, худи, тельняшка, база, топ-халтер);
расходники (базовые футболки/топы); мягкий шопер + сумка жёсткой формы; объёмный шарф в трендовом цвете, очки.

## Пропорции (норма, для чтения фигуры)
Модуль H = АС = 1/8 роста (темя→подбородок). Норма: ширина плеч = 1/4 роста; ширина груди = 1/6 роста;
ширина бёдер = ширина плеч; обхват шеи = обхват икры. Линия талии AE = 3H. Отклонения от нормы →
тип фигуры и зона коррекции. Стройность считывается по самому тонкому выигрышному месту.

## Как подключается
- `look-generator`: образы/капсула под фигуру брать ИЗ этих списков (безопасные комбинации + вещи-корректоры);
  базовая капсула при неуверенной фигуре = 8 вещей-корректоров.
- `garment-check`: оценивать вещь против правил её типа фигуры (что подчёркиваем/перекрываем).
