# architecture/reference/figure-correction/

База коррекции фигуры — knowledge base для AI-слоя (второе звено движка после цветотипов).

| Файл | Что внутри | Статус |
|---|---|---|
| `figure-correction.md` | 5 типов фигуры (особенности/задача/подходит/избегать/лучший предмет) + универсальные принципы коррекции, оптические иллюзии, цвет/ткань/фактура, вытягивание роста, вещи-корректоры | ✅ v1.0, требует валидации Ксенией |
| `body-reading.md` | Как читать фигуру по фото: модульная схема (H = 1/8 роста), точки A–K, эталонные вертикали/ширины, определение типа (ОБ−ОТ ≥ 25 → hourglass и т.д.), правила коррекции по отклонениям, кейсы Ирина/Татьяна | ✅ v1.0, требует валидации Ксенией |

## Назначение

`look-generator` при сборке капсулы проверяет каждый образ по типу фигуры; `formula-diagnostic` берёт силуэты/коррекцию в `visual_formula`. Коррекция цветом согласована с `../colortypes/` (чёрный → тёмный оттенок своей палитры).

## Источник

«Модуль 5. Типы фигур и способы коррекции» + конспекты «Силуэты», «Иллюзии», «Универсальные луки»; для `body-reading.md` — «Анализ конституции тела» + «Схема снятия плоскостных и объёмных характеристик» (`../raw-inbox/`; у последних двух текстовый слой битый, считаны постранично как изображения).

## Ключи типов

`rectangle | hourglass | pear | inverted_triangle | apple` — совпадают с `formula-diagnostic` и `look-generator`.
