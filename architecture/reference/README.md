# architecture/reference/

Справочные материалы-источники для AI-слоя и метода. Сюда складываются первоисточники (курс, тренды, палитры, фасоны) и их структурированные версии.

> **Карта покрытия и конвейер** — [`KNOWLEDGE-BASE.md`](KNOWLEDGE-BASE.md): что уже структурировано, что в очереди, куда падают новые источники (включая транскрипты аудио/видео встреч).

## Принцип: два слоя

AI-слой (`look-generator`, `style-library`, `formula-diagnostic`) **читает структурированный текст, а не PDF и картинки**. Поэтому:

- **`raw-inbox/`** — «входящие»: исходники как есть (PDF курса, скрины лукбуков, фото фасонов, тренды). Сюда можно кидать что угодно на разбор.
- **структурные `.md`** (например `colortypes/12-cvetotipov.md`) — то, что вытащено из исходников в текст и подключается к промптам. Только это «видит» AI.

Пополнение: кидаешь в `raw-inbox/` → разбираем в структурный файл → его подхватывает генератор.

## Папки

| Папка | Что внутри |
|---|---|
| `algoritmy-imidzha-kurs/` | Исходные PDF курса «Алгоритмы имиджа» (источник по ремеслу) |
| `colortypes/` | Структурный справочник 12 цветотипов (база колориста) |
| `figure-correction/` | Структурный справочник коррекции фигуры (5 типов + принципы, иллюзии, вещи-корректоры) |
| `print-mixing/` | Структурный справочник принтов и цветовых схем (6 правил микса, «принт как цвет», закон цветового пятна, акцент) |
| `prototypes/` | Библиотека прототипов + метод подбора (двойная проверка + фильтр по цветотипу) для ШАГ 5 |
| `style-typology/` | 6 чистых стилей + типажи внешности + диффузия + коммерческие/исторические — источник под 4 поля и 25 подстилей |
| `impression-lexicon/` | Дословный список ~66 характеристик анкеты (Q4/Q5) по 4 полям — основа расчёта Формулы и Identity Gap |
| `request-diagnostics/` | Диагностика запроса: конструктивный vs неконструктивный, переформулирование «про себя» до расчёта Формулы |
| `image-psychology/` | Психология имиджа: интейк-таксономия (5 блоков), ценности→стиль, имидж-проектирование (7 шагов) |
| `image-generation/` | Генерация образов: структура промпта (6 блоков), фотореалистичный тулкит, маппинг Формула→фотостиль (для preview) |
| `glossary/` | Профессиональный словарь стилиста (термины индустрии) — «наш язык» для текстов |
| `scenarios/` | Сценарные модули гардероба под образ жизни (trendy-mama — мама в декрете) |
| `raw-inbox/` | Входящие на разбор: курс, палитры, силуэты, фигуры, тренды, скрины |

## Картиночные PDF

Палитры/силуэты/фигуры в курсе — PDF из картинок (текст не извлекается). Для чтения рендерятся в PNG через PyMuPDF (`python -m pip install PyMuPDF`), затем читаются как изображения.
