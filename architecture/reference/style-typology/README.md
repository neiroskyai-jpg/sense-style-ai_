# architecture/reference/style-typology/

База стилевых типажей — knowledge base для AI-слоя. Глубокий источник под 4 поля и 25 подстилей Формулы стиля.

| Файл | Что внутри | Статус |
|---|---|---|
| `style-typology.md` | 6 чистых стилей (атрибуты) + типажи внешности (геометрия лица, макияж/причёска) + диффузия (микс) + маппинг на 4 поля/25 подстилей + коммерческие/исторические стили | ✅ v1.0, требует валидации |

## Назначение

Питает `formula-diagnostic` (распределение по полям, ШАГ 1-6), `style-library` (25 подстилей как диффузные сочетания), `look-generator` (принцип «база 2-3 стиля + точечные включения»).

## Источник

Модуль 4 «Алгоритмы имиджа»: чистые стили, диффузия, стилевые типажи, коммерческие/исторические стили (`../raw-inbox/`).

## Тонкие места

Часть коммерческих стилей (милитари, сафари, бохо, гаучо, кантри, денди, леди-лайк, морской, athleisure, спортшик, casual chic, неофутуризм) в источнике — перечислением. Детальные карточки есть для Преппи, Гранж, Минимализма/Авангарда. Остальные — на докрутку (карточка или транскрипт встречи).
