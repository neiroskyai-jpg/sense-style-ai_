# architecture/reference/style-typology/

База стилевых типажей — knowledge base для AI-слоя. Глубокий источник под 4 поля и 25 подстилей Формулы стиля.

| Файл | Что внутри | Статус |
|---|---|---|
| `style-typology.md` | 6 чистых стилей (атрибуты) + типажи внешности (геометрия лица, макияж/причёска) + диффузия (микс) + маппинг на 4 поля/25 подстилей | ✅ v1.0, требует валидации |
| `commercial-historical-styles.md` | 13 коммерческих и исторических стилей: цвета, принты, ткани, вещи, обувь, аксессуары, бренды, сезон + матрица микса и жёсткие запреты | ✅ v1.0, требует валидации |

## Назначение

Питает `formula-diagnostic` (распределение по полям, ШАГ 1-6), `style-library` (25 подстилей как диффузные сочетания), `look-generator` (принцип «база 2-3 стиля + точечные включения»).

## Источник

Модуль 4 «Алгоритмы имиджа»: чистые стили, диффузия, стилевые типажи, коммерческие/исторические стили (`../raw-inbox/`).

## Тонкие места

Карточки коммерческих и исторических стилей разобраны из модуля 4 полностью и подключены к генерации (`core/style_cards.py`). Без карточек остались подстили, которых в модуле нет: леди-лайк, морской, casual chic — на докрутку.
