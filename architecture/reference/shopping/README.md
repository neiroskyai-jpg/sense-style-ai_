# architecture/reference/shopping/

Shopping-слой — подбор реальных вещей под собранные образы. Архитектура и решение (гибрид) — `../../2026-06-09-knowledge-and-vision-architecture.md` раздел 5.

## Две фазы

- **Фаза 1 (MVP):** кураторские бренд-листы. AI выдаёт «вещь + бренд + поисковый запрос», отфильтрованные по палитре/фигуре/бюджету. Артефакт — `brand-matrix.md` (✅ v0.1).
- **Фаза 2 (апгрейд):** живой каталог (фиды Lamoda/Wildberries/Ozon), реальные SKU со ссылкой/наличием, affiliate. После первых продаж.

## Принцип

Магазин-слой подбирает вещи **под уже собранный образ**, не наоборот. Фильтр всегда: палитра цветотипа (табу нельзя) + силуэт фигуры + бюджет клиентки + стиль-сегмент. Важно: бренд привязан к **бюджету и стилевому полю**, а палитра/силуэт фильтруются на уровне **конкретной вещи** (любой бренд делает вещи всех цветов). Подключается промптом `prompts/shopping-list.md` (✅ v1.0).

> `brand-matrix.md` — **не эвергрин**: бренды/наличие меняются, ревизовать раз в сезон-полугодие.
