# Ценности и личная история → личный стиль (канон курса «Алгоритмы имиджа»)

**Тип:** knowledge base для AI-слоя. Питает `formula-diagnostic`, **чат-стилиста**, будущий
**слой личности** (Big Five). Цель: связать КТО клиентка (ценности, история, темперамент) с тем,
как это отражается в стиле — психология прежде моды. Источник: курсовая «Соответствие имиджа целям и ценностям».

## Логика: три источника → отражение в стиле
1. **Личная история** — культурная среда, наследие, привычки, среда обитания, **прожитый опыт**, **темперамент**, детство.
2. **Ценности** — что важно, каких идеалов придерживается; **модель идеального будущего** (каким видит себя/своё дело).
3. → **Отражение в стиле и личном бренде** — каждый источник «прокидывается» в визуальное решение.

Каналы связи (как читать): *прожитый опыт → ..., темперамент → ..., среда/детство → ..., модель будущего → ...*

## Пример 1. Ирина Голомаздина (креативный директор 12Storeez)
- Прожитый опыт: после инсульта ценность — время для себя, отдых, отсутствие «гонки», **внутренняя тишина** → стиль **quiet luxury** (тихая роскошь, без логомании, драмы, эпатажа).
- Темперамент: интроверт, домашний уют → **отсутствие визуального шума**, нет микса принтов/ярких цветов, нейтральная гамма, тактильные качественные ткани.
- Среда/детство: семья шила одежду с детства, уральские корни, «выход из народа» при интеллигентности → база, понятность, рациональность, зрелость, осознанность.
- Ценности: наследие, экологичность, натуральность → натуральные дорогие ткани (шёлк, кашемир, шерсть, кожа, лён).

## Пример 2. Роузи Хантингтон-Уайтли (супермодель)
- Темперамент: экспрессивный, целеустремлённый, с детства мечтала о подиуме → подиумный образ **динамичный, драматичный** (оголённые шея/ключицы/запястья/щиколотки, корсеты, кожа, вырезы).
- Среда/детство: классическая английская красота, буллинг за внешность → проактивность, критичность, культ идеального тела; образ **«современная амазонка»** (чёрный, кожа, слим).
- Модель будущего: статус, бизнес, тело мечты, расширение прав женщин → дорогие статусные бренды, тяжёлый люкс, классика + драма для подиума.
- Трансформация после 30+: для жизни — лаконичная простая одежда + большая коллекция аксессуаров/обуви/очков (**ДНК — очки-маска**), дорогие оттенки (молочный, беж, кофейный, карамельный), **тихая роскошь**, чистота линий, уважение к классике.

## Вывод (как применять)
- Один и тот же приём (тихая роскошь) у двух разных женщин закрывает РАЗНЫЕ ценности — диагностика должна объяснять стиль **через её ценности/историю**, а не как «тренд».
- Темперамент (интроверт ↔ экспрессивный) → уровень визуального шума и драмы в образе.
- Прожитый опыт и «модель идеального будущего» → главный вектор Формулы (куда ведём образ).
- Это прямой мост к слою личности: личность/ценности → желаемое впечатление → образ.
