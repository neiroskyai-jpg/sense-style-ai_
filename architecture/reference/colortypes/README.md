# architecture/reference/colortypes/

База колориста — система 12 цветотипов как knowledge base для AI-слоя.

| Файл | Что внутри | Статус |
|---|---|---|
| `12-cvetotipov.md` | Палитры 12 цветотипов: характер, база, акценты, нейтрали, металлы, табу-цвета | ✅ v1.0, требует валидации Ксенией (металлы и табу — первыми) |
| `photo-reading.md` | Как определить цветотип и контраст по фото: температура (подтоны) → контраст по 11-ступенчатой шкале → сезон → подтип (признаки 12 подтипов: глаза/волосы/кожа), природная палитра, правила портретной зоны и «1 из 2» | ✅ v1.0, требует валидации Ксенией (пороги шкалы, эталоны) |

## Назначение

Недостающее звено перед генерацией капсулы. Цветотип определяется → его палитра жёстко идёт в `visual_formula.palette`, табу-цвета — в `stop_list`. `look-generator` не имеет права перекрыть табу цветотипа.

Подключается так же, как `architecture/prompts/style-library.md` — подклеивается в системный промпт.

## Источник

Палитры курса «Алгоритмы имиджа» (картиночные PDF в `../raw-inbox/`), считаны со swatch-сеток. Базовые/акцентные цвета — прямо с палитр; металлы и табу достроены по 12-сезонной логике и сверены с подтверждённым эталоном Натурального Лета.

Для `photo-reading.md` — «Директный классический метод цветотипирования» + Модуль 3 Урок 4 «Современный подход к цветотипированию, контрастность внешности» (там же; текстовый слой битый, считаны постранично как изображения).
