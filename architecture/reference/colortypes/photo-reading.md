# Чтение колорита по фото — метод определения цветотипа и контрастности

Как AI (Vision) определяет цветотип из 12 и уровень контрастности по фото клиентки. Дополняет `12-cvetotipov.md` (там — палитры и табу, здесь — КАК определить). Источники: «Директный классический метод цветотипирования» + Модуль 3 Урок 4 «Современный подход к цветотипированию, контрастность внешности» (курс «Алгоритмы имиджа»).

## Порядок определения

1. **Температура** (тёплая / холодная) — по подтону кожи, оттенку волос, пигменту глаз.
2. **Контрастность** (высокая / средняя / низкая) — разница светлоты между самым тёмным и самым светлым тоном внешности.
3. **Чистота** (чистый / приглушённый) и **светлота** (светлый / тёмный) — уточняют сезон.
4. Сезон по квадранту → подтип по таблице признаков (12 вариантов) → палитра из `12-cvetotipov.md`.

Оценивать **текущее состояние**: загар, окрашивание волос и макияж меняют и температуру восприятия, и уровень контрастности. Если волосы явно окрашены — температуру читать по коже и глазам, контраст считать по фактическому цвету волос (клиентка носит его сейчас).

## Шаг 1. Температура

| Признак | Холодная внешность | Тёплая внешность |
|---|---|---|
| Кожа | розовый, сиреневый, голубой подтон; оливковый тон | персиковый или желтоватый подтон, без синевы; веснушки |
| Волосы | пепельный, холодный оттенок (брюнетки, русые, блондинки) | золотистый оттенок, жёлтый подтон, рыжеватые пряди |
| Глаза | серые, голубые, леденяще-синие, тёмно-коричневые, чёрные | серые/голубые с жёлтыми или зелёными вкраплениями; зелёные, карие, медовые; «хамелеоны» |

Подтоны кожи точнее: холодные — серо-бежевый, нейтрально-бежевый, лилово-бежевый; нейтральные — коричнево-бежевый, розово-бежевый, зелёно-бежевый; тёплые — жёлто-бежевый, персиково-бежевый, оранжево-бежевый.

**Оговорки (частые ошибки):** больше всего пигмента — в радужке глаза, но голубые глаза ≠ автоматически холодный тип, зелёные ≠ тёплый. Температуру решает совокупность, в первую очередь подтон кожи.

## Шаг 2. Контрастность

Контрастность = соотношение самых светлых тонов внешности с самыми тёмными (волосы / глаза / кожа).

**Метод шкалы:** ахроматическая шкала из 11 делений (1 = чёрный, 11 = белый). Перевести самый тёмный и самый светлый тон внешности в деления, посчитать разницу:

| Разница делений | Уровень | Эталоны |
|---|---|---|
| 7–10 | высокая | Моника Беллуччи, Энн Хэтэуэй (тёмные волосы + светлая кожа) |
| 4–7 | средняя | Кейт Миддлтон, Анджелина Джоли, Наталья Водянова |
| 0–4 | низкая | Кэмерон Диас, Тильда Суинтон, Светлана Ходченкова |

Для Vision: мысленно перевести фото в ч/б — насколько далеко разнесены волосы, кожа и глаза по светлоте.

## Шаг 3–4. Квадрант сезонов и 12 подтипов

Оси квадранта: светлый/тёмный × тёплый/холодный; диагонали — чистый/приглушённый.

| Сезон | Формула | Эталон |
|---|---|---|
| Весна | чистый, тёплый, светлый | Кейт Босуорт, Риз Уизерспун, Аманда Сейфрид |
| Лето | светлый, холодный, приглушённый | Дженнифер Энистон, Сара Джессика Паркер, Милла Йовович |
| Осень | приглушённый, тёплый, тёмный | Джулия Робертс, Эмма Стоун, Дженнифер Лопес, Бейонсе |
| Зима | тёмный, холодный, чистый | Лив Тайлер, Энн Хэтэуэй, Деми Мур, Сальма Хайек |

### Дополнительные признаки сезона (структура кожи и волоса)

- **Зима:** кожа плотная, хорошо загорает, без веснушек и капилляров; волос плотный, густой; брови/ресницы чёрные; белая эмаль зубов и белый белок глаза; глаза непрозрачные.
- **Весна:** кожа тонкая, прозрачная, склонна к красноте и шелушениям, обгорает; глаза прозрачные; брови/ресницы русые; губы персиково-розовые. Образ «девочка в пшеничном поле».
- **Лето:** кожа тонкая, сероватая, с голубым подтоном, частые покраснения; веснушки серые; волосы «мышиные», пепельные; губы розовые без персика. Самый комплиментарный цвет — голубой.
- **Осень:** кожа с персиково-оранжевым подтоном, краснота редка; веснушки рыжие; волос плотный, рыжий/каштановый; брови рыжие/коричневые.

### ПРАВИЛО ПРИОРИТЕТА: контраст важнее таблицы (валидировано Ксенией, кейс Марии)

При выборе ПОДТИПА внутри сезона (natural / contrast / light) **уровень контраста по шкале (Шаг 2) главнее табличных цветовых признаков** глаз/волос/кожи. Таблица выше — вспомогательная (помогает с сезоном и описанием), но НЕ перебивает контраст.

Зачем: на пограничных лицах таблица и контраст конфликтуют. Пример — Мария: голубые глаза + каштановые волосы + фарфоровая кожа по таблице = «зима контрастная», НО её фактический контраст средний → подтип **winter_natural** (подтверждено Ксенией). Светлые глаза при тёмных волосах создают видимость контрастности, но если по 11-ступенчатой шкале разница средняя — это natural, не contrast. Решает измеренный контраст, не цвет глаз.

Порядок: сезон (квадрант) → контраст по шкале определяет подтип → таблица только уточняет/подтверждает. При расхождении побеждает контраст; второго кандидата по таблице — в `colortype_alternative`.

## Природная палитра (современный подход)

Жёсткая привязка «человек = сезон» — вспомогательная рамка. Рабочий слой — индивидуальная палитра, считанная с внешности: пигменты радужки, губ, бровей, волос, эмали зубов = **цвета природной цветовой гармонии**. Они же — нейтральные цвета клиентки, основа базового гардероба (до 70–80% цветов).

Для Vision это значит: помимо ярлыка подтипа, снять с фото 5–7 конкретных пигментов (волосы у корней, радужка, кожа в тени и на свету, губы) и отдать их в `visual_formula.palette` как персональные нейтрали — поверх палитры подтипа из `12-cvetotipov.md`, не вместо неё.

**Как это делает человек на курсе (и что повторяет наш Vision):** стилист фотографирует клиентку без макияжа у окна (портретная зона), прогоняет через **Adobe Capture** — приложение вытягивает палитру пигментов с лица. Лайфхак курса: макросъёмка радужки глаза даёт расширенную палетку (в радужке больше всего пигмента). Наш Vision — это **автоматизированный Adobe Capture**: тот же сэмплинг пигментов с фото, только сразу с привязкой к сезону/подтипу и палитре `12-cvetotipov.md`. Отсюда же требование к UX фото-шага: без макияжа, свет от окна — иначе пигменты считаются неверно (макияж и тёплый свет искажают подтон, ровно как искажают и для человека-стилиста).

## Практические правила (для look-generator и Style Book)

1. **Портретная зона решает всё.** Подходящие цвета определяются только зоной у лица; остальные цвета образа обязаны сочетаться с портретной зоной. Любимые цвета вне палитры — «уводить вниз»: низ, обувь, аксессуары.
2. **Контраст внешности = контраст одежды.** Чем контрастнее внешность, тем ярче, чище и контрастнее цвет у лица. Чем мягче внешность — тем приглушённее оттенки и без резких контрастов.
3. **Правило «1 из 2».** Цвет в портретной зоне обязан соблюдать минимум один из двух параметров — температуру или насыщенность. Пример: светлое лето → у лица холодные И/ИЛИ приглушённые оттенки; яркий тёплый — нарушает оба, запрещён.
4. Изменение цвета волос рушит природную контрастность (пример: Ким Кардашьян в карамельном блонде теряет высокий контраст) — рекомендации давать под фактический цвет волос.

## Как подключается

- `formula-diagnostic` ШАГ 7: определение цветотипа и контраста по этому файлу → палитра/металлы/табу из `12-cvetotipov.md`.
- Будущий `vision-analyzer.md`: этот файл — ядро инструкции «что читать с фото» по цвету.
- Подбор прототипа (ШАГ 5): уровень контрастности — обязательная ось сравнения (контраст важнее сезона).
- В клиентских текстах ярлык сезона не используется — см. «Бережный язык» («холодный подтон + средняя глубина + высокий контраст», не «цветотип Зима»).
