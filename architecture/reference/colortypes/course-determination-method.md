# Метод определения цветотипа (канон курса «Алгоритмы имиджа», Модуль 3 Урок 4)

**Тип:** knowledge base для AI-слоя. ПЕРВИЧНЫЙ источник по ОПРЕДЕЛЕНИЮ цветотипа —
питает `vision-analyzer` (как читать фото) и `formula-diagnostic`. Источник: курсовая
презентация «Современный подход к цветотипированию. Контрастность внешности» (image66).

**Зачем:** vision-модель путала тёплый↔холодный (осень↔зима — обе глубокие, но разной
температуры). Здесь — точные критерии: сначала ТЕМПЕРАТУРА, потом СВЕТЛОТА → сезон;
КОНТРАСТ → подтип.

## Порядок определения (строго)

1. **ТЕМПЕРАТУРА** кожи: холодная / тёплая (главный, самый частый промах).
2. **СВЕТЛОТА/глубина** внешности: светлая ↔ глубокая.
3. → СЕЗОН (4 гаммы). 4. **КОНТРАСТ** кожа↔волосы → подтип (контрастная/натуральная/светлая).

## 1. ТЕМПЕРАТУРА (тёплый ↔ холодный) — определяющий признак

**ХОЛОДНАЯ внешность:**
- Волосы: пепельные, холодного оттенка (брюнетки, русые, блондинки).
- Глаза: серые, голубые, ледяные синие, красноватого/тёмно-коричневого оттенка либо чёрные.
- Кожа: преобладание розового, сиреневого, голубого; оливковый тон.

**ТЁПЛАЯ внешность:**
- Волосы: золотистый оттенок, жёлтый подтон, рыжеватые пряди.
- Глаза: серые/голубые с жёлтыми или зелёными вкраплениями, зелёные, карие, медовые; меняют оттенок при свете.
- Кожа: персиковый/желтоватый оттенок, БЕЗ холодного синего подтона; часто веснушки.

**Важно (частые ошибки):** голубой глаз ≠ обязательно холодный; зелёный глаз ≠ обязательно тёплый.
Больше всего пигмента — в **радужке глаза**; ориентироваться на общий подтон кожи + волос, не на один признак.

## 2. КОНТРАСТНОСТЬ ВНЕШНОСТИ

Соотношение самых светлых и самых тёмных тонов внешности (разница цвета волос, глаз, кожи)
по 11-ступенчатой ахроматической шкале (1 чёрный … 11 белый):
- разница **0–4 деления → НИЗКАЯ** (мягкая) контрастность;
- разница **4–7 → СРЕДНЯЯ**;
- разница **7–10 → ВЫСОКАЯ**.

Меняют уровень контраста: загар, оттенок волос, макияж. Правило: **чем контрастнее внешность,
тем ярче цвет в портретной зоне**; низкий контраст → приглушённые оттенки без резких контрастов.

## 3. ЧЕТЫРЕ СЕЗОНА — критерии (кожа/волосы/брови/глаза/губы)

**ЗИМА** (холодная, глубокая): кожа с голубым/серым пигментом, оливковая, смуглая; плотная,
хорошо загорает, без веснушек/капилляров; волосы тёмные почти чёрные или шоколадные, плотные;
брови/ресницы чёрные густые; глаза тёмно-карие, чёрные или ярко-голубые (непрозрачные); очень
белая эмаль зубов и белок. Оттенки: холодные, яркие/насыщенные/неоновые, контрасты (ч/б).

**ВЕСНА** (тёплая, светлая): кожа тёплая персиково/жёлтая, тонкая, прозрачная, склонна к красноте,
веснушки, обгорает; волосы блонд (светлый/тёмный/соломенный), русый; брови/ресницы светлые;
глаза прозрачные зелёные/карие/серые/голубые/серо-зелёные; губы персиково-розовые. Оттенки:
нежные, свежие, прозрачные; сочные, чистые, яркие.

**ЛЕТО** (холодная, светлая): кожа холодная, очень серая, голубоватый подтон, серые веснушки,
тонкая, склонна к покраснениям; волосы тёмно-русый (серый), мышиный, пепельный блонд; брови/ресницы
светлые; глаза светлые (серые, серо-голубые, голубые); губы розовые, БЕЗ персикового пигмента.
Оттенки: мягкие, приглушённые, без сильного контраста, с серым подтоном, «выцветшие»; голубой — самый комплиментарный.

**ОСЕНЬ** (тёплая, глубокая): кожа с тёплым персиково-оранжевым/жёлтым подтоном, редко краснота,
могут быть веснушки; волосы рыжие (светло/тёмно), каштановые, плотные; брови/ресницы русые/рыжие/коричневые;
глаза зелёный/чайный/серо-зелёный/медовый/карий/тёмно-карий. Оттенки: глубокая сложная палитра —
жёлтый, оранжевый, бордо, красный, изумрудный, коричневый, шоколадный, кремовый, молочный.

> **Карта температуры × светлоты:** тёплые = ВЕСНА (светлая) + ОСЕНЬ (глубокая); холодные = ЛЕТО
> (светлая) + ЗИМА (глубокая). Осень и зима ОБЕ глубокие — различай по ТЕМПЕРАТУРЕ (осень тёплая, зима холодная).

## 4. Принципы подбора цвета (как внедрять «свои» цвета)

- **Природно-цветовая гармония = нейтрали:** пигмент из природного цвета глаз/губ/бровей/волос/кожи
  максимально подходит; нейтрали = основа базового гардероба, **70–80% всех цветов**.
- Цвета подходят/не подходят зависят ТОЛЬКО от **портретной зоны** (у лица); остальной образ
  сочетается с тем, что у лица.
- Если «любимые» цвета диссонируют с природной палитрой — у лица природную, любимые уводим вниз.
- Правило температуры и насыщенности: при холодной низкоконтрастной внешности (светлое лето) у лица —
  холодные приглушённые без контрастов; минимум 1 из 2 параметров (температура ИЛИ насыщенность) держать.

## Как подключается
- `vision-analyzer`: определять colortype по ЭТИМ критериям — сначала температура (тёплый/холодный),
  потом светлота → сезон; контраст кожа↔волосы → подтип. Точные оттенки палитр — `12-cvetotipov.md`.
- `formula-diagnostic`/RAG: грунтовать диагностику этим методом. Измеренный контраст (`core/colortype.py`)
  ставит подтип; этот документ — про ТЕМПЕРАТУРУ и СЕЗОН (то, что было слабым местом).
