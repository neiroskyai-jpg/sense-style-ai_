# Источники насмотренности — справочник (кино, книги, каналы, аккаунты)

Конкретный контент для развития насмотренности. Дополняет `inspiration-method.md` (метод «как смотреть и анализировать») списком «что смотреть». Источник: «ГИД по насмотренности» курса «Алгоритмы имиджа».

**Зачем нам:** культурный слой для AI. Эстетика десятилетий и фильмов = референсы для `image-generation` (визуальный язык образа), для контента (email/посты могут опираться на узнаваемую киноэстетику), для подбора прототипов. Аккаунты/каналы — медленно меняются, ревизия раз в год.

## Как подключается

- `image-generation` — киноэстетика десятилетий как визуальный референс промпта (свет, палитра, настроение эпохи).
- Контент-слой (email/посты) — опора на узнаваемую эстетику вместо абстракций.
- `inspiration-method.md` — метод работы с этими источниками (анализ «что зацепило / почему», 15-20 мин показов в день).
