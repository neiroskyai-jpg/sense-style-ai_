## Назначение

Чтобы тексты `look-generator`/`style-book` звучали как стилист, а не как нейросеть. Термины — для точности и тона. **Важно:** по tone of voice мы объясняем, а не сыпем жаргоном — клиентке термин переводим на человеческий.
