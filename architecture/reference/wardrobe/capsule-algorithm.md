# Алгоритм создания капсулы (канон курса «Алгоритмы имиджа»)

**Тип:** knowledge base для AI-слоя. Питает `look-generator` (сборка капсулы/образов) и Карту стиля.
Источник: курсовая «Алгоритм создания капсулы» (А. Ермышева). Грунтует капсульную логику
«мало вещей → много образов» ([[universal-looks-by-figure]] — что брать под фигуру).

## Базовые принципы
- Капсула = **6–12 вещей**, которые идеально сочетаются между собой.
- Каждую НОВУЮ вещь берём так, чтобы она миксовалась **минимум с 3–4** уже имеющимися.
- **Разные фасоны** (асимметрия, деконструкция, сложный крой) — избегаем однообразия.
- Капсула ≠ базовый гардероб: добавляем **трендовые цвета, must-have/must-need сезона**.
- Начинаем с **ровных однотонных** вещей, постепенно усложняем (принты, трендовые цвета).

## Математика капсулы (ценность = число комбинаций)
4 вещи → ~14 рабочих комбинаций; 6 вещей → ~30; деловая 9 вещей (4 низа+5 верхов) → «4+5=42».
Чем больше взаимосочетаемость, тем выше cost-per-wear каждой вещи. Это и продаём: не вещи — систему.

## Тип 1. Кэжуал (повседневная), 10–11 вещей
1. Трикотажный костюм (юбка/брюки) вместо строгого брючного.
2. 1 идеально сидящие джинсы + 1 принципиально другой низ (мягкая струящаяся юбка в бельевом стиле).
3. 5 разноплановых верхов (рубашка из вискозы, белая хлопковая рубашка, футболка с акцентным принтом, жакет букле/твид, кардиган).
4. Верхний слой — 1 актуальная шуба/дублёнка.
5. Обувь: 2 пары — 1 элегантная на каблуке (остроносые ботильоны) + 1 мягкая комфортная без каблука (угги/лоферы/челси/броги/кроссовки).
6. 1 актуальная сумка (такса/багет, мягкий шопер, кросс-боди).
7. Расходники (майки в рубчик, топы, кроп-топы, боди) + аксессуары.

## Тип 2. Деловая «4+5=42» (4 низа, 5 верхов, 9 предметов)
1. 1 актуальный цельный однотонный костюм (брючный/с юбкой/тройка).
2. 2-й костюм — принципиально другой по крою/фасону/комплектности (можно с принтом/юбкой).
3. Независимые 1 верх + 1 низ, сочетающиеся между собой (юбка экокожа + блуза шифон).
4. Добавить **ахроматы** — максимально нейтральные базовые вещи (вписать по оттенку/функции).
5. Доработать интересными верхами (+3: кардиган, свитер, блуза).
- **Принты для делового:** геометричный, равномерные паттерны, мелкий принт «как визуальный шум».

## Тип 3. Универсальная «и в пир, и в мир» (классика + кэжуал)
1. 1 актуальный брючный костюм. 2. 2 разноплановых низа. 3. 3–4 актуальных верха.
4. Верхняя одежда (разноплановая). 5. 2 сумки разной формы. 6. 2 пары обуви. 7. Аксессуары + расходники.

## Как подключается
- `look-generator`: собирать капсулу по этому алгоритму — ядро (костюм/база) → разноплановые низы →
  разноплановые верхи → ахроматы-нейтрали → верхний слой → 2 обуви/2 сумки → расходники+аксессуары;
  каждая вещь миксуется с ≥3–4. Под фигуру — безопасные вещи из [[universal-looks-by-figure]].
- Карта стиля: «combination_count» = число комбинаций (по математике выше) — показываем ценность системы.
