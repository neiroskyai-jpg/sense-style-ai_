# architecture/reference/wardrobe/

База методологии гардероба — структура рационального и базового гардероба как фундамент капсулы. Эвергрин.

| Файл | Что внутри | Статус |
|---|---|---|
| `rational-wardrobe.md` | 8 принципов рационального гардероба, формула пропорций (база 70-80% / тренд 20-30% / аксессуары 1/3), критерии базовой вещи, состав базовой капсулы, цвет базы по зонам внешности, CPW + чек-лист ревизии, диаграмма занятости | ✅ v1.0, требует валидации Ксенией |

## Назначение

`look-generator` собирает капсулу как **база (этот файл) + тренд (`../trends/`)**. Доля базы/тренда, критерии базовой вещи, привязка нейтрали к зонам внешности — отсюда. Чек-лист ревизии и CPW — для будущего `wardrobe-analyzer`.

## Источник

«Рациональный гардероб / Базовый гардероб» (Анастасия Ермышева, курс «Алгоритмы имиджа»; имя файла в `raw-inbox` — «База. Тренды», текстовый слой битый, считано постранично как изображения).

## Граница

Структура гардероба и доля базы — здесь. Конкретные тренды сезона — `../trends/`. Алгоритм сборки капсулы (правило связности, число комбинаций) — промпт `look-generator`. Коррекция по фигуре — `../figure-correction/`.
