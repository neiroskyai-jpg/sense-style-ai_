# Сценарий «Домашний гардероб»

**Тип:** knowledge base для AI-слоя (сценарный модуль). Подключается к `look-generator`, когда у клиентки высокая доля «дом» в `context.time_distribution` (по диаграмме занятости из `wardrobe/rational-wardrobe.md` дом часто 40-70% времени, особенно у мам в декрете и на удалёнке) — и эта сфера почти всегда недооценена в гардеробе.

**Зачем именно нам:** дом — самая большая по времени и самая «забытая» зона. Закрыть её = закрыть разрыв «дома я никто / расхлябанная» в точке перехода. Связан со сценарием [[trendy-mama]] (дом-капсула из 3 блоков).

**Источник:** «ДОМАШНИЙ ГАРДЕРОB» школы «Алгоритмы имиджа» (raw-inbox, считано постранично как изображения).

**Статус:** v1.0. Бренды — обновляемая/валидируемая часть.

---

## Основная задача

Создание и поддержание **внутреннего комфорта и расслабленности**. Домашняя одежда — для тела: дышащие материалы, не сковывает движений, по сезону (лето — топы/шорты; зима — пижамы/пушистые тапочки). Комфорт не равен «расхлябанности»: даже домашнее стилизуется.

## Три функциональных блока

1. **Одежда для дома и сна** — удобная, из дышащих материалов, не сковывает движения.
2. **Одежда для спорта** — под конкретный вид спорта, строго по размеру.
3. **Одежда для встречи гостей** — умеренно нарядная, без излишней откровенности (можно показаться мужу/гостям, не переодеваясь в «выход»).

## Из чего строить (по блокам)

**Дом и сон:** пижамы (вискоза, шёлк, хлопок, мягкая шерсть) · ночные сорочки · топы, брюки, шорты, рубашки · джемперы из шерсти и кашемира · худи, толстовки · халаты.

**Спорт:** топы, футболки, рашгарды · шорты, легинсы, теннисные юбки · лонгсливы, спортивный кардиган · комбинезоны. (Подробная спорт-логика для мам — в [[trendy-mama]].)

**Встреча гостей:** пижамные костюмы из шёлка/вискозы/хлопка · мягкие широкие брюки, юбки из трикотажа · лонгсливы, рубашки, джемперы · длинные кардиганы · мягкие платья.

## Примеры стилевых сочетаний (как «застилизовать» домашнее)

- Шёлковый халат + сорочка.
- Удлинённый кардиган + топ и трусы-боксеры (домашний слой).
- Пижамный костюм с шортами + высокие носочки + мягкие пушистые тапочки.
- Топ и юбка из мягкого трикотажа + верхний слой для тела.
- Кардиган из кашемира + шёлковые брюки.
- Пижамный комплект + объёмная укладка + минималистичные украшения.
- Летнее летящее платье + небрежная укладка.
- Платье-комбинация с кружевом + украшения + красная помада (для гостей/мужа).

> Приём: домашняя капсула собирается так, чтобы один-два штриха (укладка, украшения, кардиган, помада) переводили её из «для сна» в «для гостей» без переодевания.

## Капсула домашнего гардероба (пример)

Из минимума — максимум: пижамный костюм · сорочка/топы · кардиган (кашемир/трикотаж) · мягкие широкие брюки · халат · пушистые тапочки + кроссовки (спорт-блок) · мягкое платье. Монохром в нейтрали цветотипа (пример курса — пыльно-розовая и серая капсулы) даёт сочетаемость всего со всем.

## Привязка к движку

- **Цвет/фигура/формула** — из своих баз (`colortypes`, `figure-correction`, формула клиентки). Домашнее не выпадает из цветотипа: нейтрали те же, что в базе (`wardrobe/rational-wardrobe.md`).
- **Доля капсулы** — по диаграмме занятости (`time_distribution.home`): чем больше дом, тем весомее эта капсула в гардеробе.
- **Бренды (РФ, фаза 1 шопинга, требуют валидации):** дом/сон — Belle You, Cozy Home, Kuchenland, Zarina, Lime, Sela, Mark Formelle, Women’secret, Tezenis, Incanto, Intimissimi; спорт — Bonafide, Reebok/Adidas/Nike, Euphoria, Figura Activewear; гости — Poetique, 12Storeez, Present&Simple, Zarina, Lime. Маркетплейсы — Lamoda/WB. (Свести с `shopping/brand-matrix.md` при валидации.)

## Как подключается

`look-generator` подключает этот сценарий при высокой доле `home` в time_distribution — добавляет дом-капсулу (3 блока) к основной капсуле. Бережный язык: дом — это «зона восстановления и себя для себя», не «расхлябанность». Цвет/фигуру/формулу берёт из своих баз.
