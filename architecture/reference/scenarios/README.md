# architecture/reference/scenarios/

Сценарные модули гардероба под образ жизни клиентки — knowledge base для AI-слоя. Подключаются к `look-generator` по доминанте жизненного сценария.

| Файл | Когда подключается | Статус |
|---|---|---|
| `trendy-mama.md` | Материнство/маленькие дети, запрос на комфорт и спорт-шик (высокая доля «дом/семья/дети») | ✅ v1.0 |
| `home-wardrobe.md` | Высокая доля «дом» в time_distribution (декрет, удалёнка) — самая большая и забытая зона. 3 блока: дом/сон, спорт, встреча гостей + капсула | ✅ v1.0 |

## Назначение

Сценарий задаёт функциональные требования и логику капсулы (комбинаторика, многослойность, обувь, дом-капсула), а цвет/фигуру/принт/формулу берёт из своих баз. Так капсула учитывает реальную жизнь клиентки, а не только её Формулу стиля.
