# architecture/reference/print-mixing/

База принтов и цветовых схем — knowledge base для AI-слоя (третье звено движка после цветотипов и коррекции фигуры).

| Файл | Что внутри | Статус |
|---|---|---|
| `print-mixing.md` | 6 правил микса принтов + «принт как цвет» + цветовые схемы (монохром, колорблок, растяжка, комплементарная/аналоговая/триада) + закон цветового пятна (пропорции) + цветовой акцент | ✅ v1.0, требует валидации Ксенией |

## Назначение

`look-generator` при вводе принта в капсулу проверяет цвета принта по цветотипу, называет основание связи при миксе 2+ принтов, держит один цветовой акцент и неравные пропорции. Принт = цвет, поэтому всё согласовано с `../colortypes/` (табу-цвета нельзя и внутри рисунка) и `../figure-correction/` (активный принт — туда, где «мало»).
