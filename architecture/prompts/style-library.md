# style-library.md — справочник атрибутов стилей для AI-слоя

**Тип:** knowledge base (не системный промпт). Это справочник, который подключается к промптам генерации — `look-generator.md` и `wardrobe-analyzer.md`. Сам по себе JSON не возвращает.

**Источник правды по таксономии:** [`sense-style-method.md`](../sense-style-method.md) (4 базовых стиля + 25 подстилей). Этот файл **не вводит новых стилей** — он только наполняет уже существующие подстили жёсткими атрибутами: цвета, принты, ткани, ключевые вещи, бренды-референсы и правила микса.

**Источник данных:** учебник «Алгоритмы имиджа», модуль «Основные коммерческие и исторические стили» (17 стилей) + глоссарий терминов. Где таксономия школы расходится с нашей — выигрывает `sense-style-method.md`.

> ⚠️ Если методология меняется — синхронизировать этот файл. Версия: v1.0 (требует валидации Ксенией).

## 1. Универсальные правила микса (применяются ВСЕГДА при генерации образа)

Это hard-rules для AI. Нарушение = брак образа.

1. **Не больше 2 стилей в образе, максимум 3 точечно.** Чем лаконичнее и чище образ — тем лучше. Перестилизация («ёлка») — главная ошибка.
2. **Современное прочтение, а не цитирование «в лоб».** Прямое цитирование исторического стиля (полный преппи, полный милитари, полный денди) = образ для тематической вечеринки/фотосессии, не для жизни. Берём знаковые элементы стиля + миксуем с базой (классика, кэжуал, минимализм).
3. **Родственные стили миксуются свободно, неродственные — осторожно.** Группы родства — в разделе 4.
4. **Авангард и интеллектуальный минимализм — стилевая чистота.** Эти два НЕ мешаются ни с романтикой, ни с драмой (спортшик/кэжуал-шик), ни с тяжёлым люксом (Gucci, Prada, Balmain, Dior, Louis Vuitton). Образ выдержан строго в стиле.
5. **Рустик/деревенька не сочетается с драмой и с бохо** (бохо — «на богатом», рустик — про простоту; конфликт уровней).
6. **Неофутуризм в чистом виде не адаптивен для жизни** — только отдельные элементы, микс с классикой/спортом/денимом.
7. **Деконструктивизм — это приём, а не наполнение образа.** Лёгкая «сломанная» форма, асимметрия, незавершённый край. Точечно.

## 3. Атрибуты по стилям

Формат каждой карточки: **Цвета · Принты · Ткани · Ключевые вещи · Бренды-референсы · Миксуется с · Стоп.**
Бренды даны как референс эстетики (для шопинг-листа подбираем доступные аналоги по сегменту клиентки).

### Преппи (академическая классика)
- **Цвета:** тёмно-синий, беж, молочный, бордо, изумрудный, серый, чёрный, белый; в современном прочтении — яркие акценты (лимонный, фуксия).
- **Принты:** клетка (тартан, гленчек, ромб-аргайл), тонкая меловая полоска, полоска-колледж, гербы/символика, морской принт.
- **Ключевые вещи:** оксфордские рубашки, трикотаж (жилет ромб-аргайл, джемпер, поло, регби), клубный пиджак, мини-юбка в складку, чиносы, прямые тёмные джинсы.
- **Обувь:** лоферы, ретро-кеды, грубые ботинки/мартинсы, жокейские сапоги.
- **Бренды-референсы:** Miu Miu, Ralph Lauren, Lacoste, Celine, Prada, Tommy Hilfiger.
- **Миксуется с:** кэжуал, спорт, морской, английский загородный, охотничий, athleisure, чистая классика, гранж, 70-е/00-е, бохо, гаучо, этно.
- **Стоп:** полное цитирование «в лоб» (форма школьника на корпоратив).

### Денди-Гарсон (маскулинная классика)
- **Цвета:** приглушённые (серый, серо-синий, бордово-коричневый, сливовый, песочный, горчичный, болотный) + чёрный/белый + точечные яркие акценты (жёлтый, синий).
- **Ткани:** высококачественная шерсть, смесовые.
- **Ключевые вещи:** смокинг, костюм двойка/тройка, двубортный костюм, белоснежная сорочка, брюки мужского покроя, жилет, приталенное пальто, юбка-карандаш.
- **Обувь:** дерби, оксфорды, лоферы, монки, броги, жокейские сапоги.
- **Аксессуары:** галстук, шейный платок, шляпа/кепи/берет, запонки, подтяжки, перчатки.
- **Бренды-референсы:** Saint Laurent, Tom Ford, Armani, Dries Van Noten, Marc Jacobs, Dolce & Gabbana.
- **Главное правило:** всё сидит точно по фигуре, тщательно отглажено.
- **Миксуется с:** классика, минимализм, кэжуал, драма (точечно).

### Минимализм (интеллектуальная классика)
- **Цвета:** база (белый, чёрный, серая гамма, глубокий синий/коричневый, приглушённые красные/зелёные), холодные и приглушённые, без логомании и принтов.
- **Ткани:** максимально дорогие и качественные — шерсть, шёлк, атлас, сатин, кожа, плотный гладкий трикотаж.
- **Принципы:** «дорогая простота», функциональность, чистота форм, геометрия/архитектурность, многослойность как конструкция.
- **Бренды-референсы:** Max Mara, COS, Jil Sander; миддл — Ushatava, Gate 31, Asya Semenova.
- **Стоп (критично):** НЕ мешать с романтикой, драмой (спортшик/кэжуал-шик), тяжёлым люксом. Стилевая чистота.

### Авангард (творческая классика)
- **Цвета:** сложные глубокие (шоколад, кофе с молоком, припылённый зелёный, ржаво-кирпичный, мокко) + белый/беж/хаки/молочный/серый/синий/чёрный.
- **Крой:** сложный архитектурный, чистые линии, асимметрия, игра объёмов, закрытое тело (нет секса/прилегания — нет примеси драмы).
- **Принты:** отсутствуют либо графичные/интеллектуальные/statement.
- **Аксессуары/обувь:** statement-украшения, небанальная обувь (Tabi, меховые тапки, рафия/сетка).
- **Бренды-референсы:** Old Celine (эпоха Фиби Файло), The Row, Lemaire, Toteme, Jil Sander, Proenza Schouler, Khaite, Acne.
- **Стоп:** романтика, драма, тяжёлый люкс. Строгая стилевая чистота.

### Гранж
- **Цвета:** тёмные драматичные + кислотные (чёрный, красный, бордо, графит, антрацит, бургунди, ядовито-жёлтый, ярко-зелёный, тёмно-синий).
- **Принты:** леопард, тартан (жёлто-/красно-/сине-чёрный), пейсли ч/б.
- **Приёмы:** нарочитая небрежность, потёртости/рванина, многослойность (рубашка на бёдрах), унисекс/маскулинность, контраст объёмов (широко-узко-широко).
- **Ключевые вещи:** рваный деним, косуха, лаковая/грубая кожа, оверсайз-лонги с группами, тартан-рубашки, бельевые платья с кружевом.
- **Обувь:** мартинсы, ботфорты, конверсы/вансы, казаки, чопперы.
- **Современное прочтение:** гранж + минимализм, лаконичность 90-х (без перегруза БДСМ-атрибутикой).
- **Миксуется с:** минимализм, классика, преппи.

### Леди-лайк (элегантная романтика)
- См. атрибуты Романтики в `sense-style-method.md` §4.3. Сдержанная женственность, чистые силуэты, элегантность.

### Бохо (этника + романтизм + наив)
- **Цвета:** обилие сложных богатых (коричневый, бордо, марсала, глубокий красный, шоколад, кэмел, ириска, изумруд).
- **Принты:** этнические — мильфлёр, пейсли, ориентальные, мелкая рябь.
- **Ткани:** натуральные дорогие — шёлк, замша, кожа, вельвет, шерсть, сложная ручная вязка. **Отличие бохо:** роскошный, богемный, дорогой (в отличие от гаучо/кантри/этно).
- **Приём:** многослойность, летящий/струящийся крой, асимметрия, макси, обилие декора (бахрома, вышивка, макраме, кружево, натуральные камни).
- **Бренды-референсы:** Zimmermann, Isabel Marant, Saint Laurent, Loewe, Chloé, Gucci, Etro.
- **Миксуется с:** минимализм, классика (деловой), драма (гламур), родственные (натуральный, гаучо, кантри).
- **Стоп:** рустик (конфликт «богато/просто»).

### Гаучо · Кантри/вестерн (натуральный, этно)
- **Цвета:** природная приглушённая гамма (ореховый, медовый, песочный, болотный, рыжий, охра, коричневый, оттенки зелёного).
- **Принты:** геометрия + растительные орнаменты, коровий принт, клетка, мелкая полоска.
- **Ткани:** деним, хлопок, лён, кожа, замша, вельвет.
- **Вещи:** широкие брюки (бомбачи)/кюлоты, пончо, замша с бахромой, свободные рубашки (часто со шнуровкой), юбка-солнце многоярусная, комбинезоны, деним.
- **Декор:** бахрома, заклёпки, шнуровка, массивные пряжки, кожаные браслеты.
- **Бренды-референсы:** Ralph Lauren, Hermès, Isabel Marant, Chloé, Gucci, Comme des Garçons.
- **Миксуется с:** классика, сафари, бохо, этно.

### Милитари · Сафари (натуральный, утилитарный)
- **Цвета:** камуфляж и его оттенки (оливковый, беж, хаки, кремовый, серый), синий, чёрный; сафари — природно-песочная гамма + хищные принты (леопард, питон, зебра).
- **Ткани:** плотные формодержащие, только натуральные (шерсть, износостойкий хлопок/канвас, твид, лён), для сафари — «дышащие».
- **Вещи:** брюки/юбки карго, бермуды, пиджаки, пальто/бушлат/шинель, тренч, M-65, жилет Вассермана (много карманов), комбинезоны, анорак.
- **Аксессуары:** сумки-почтальоны, поясные, ремни с пряжками, шляпы/панамы/повязки.
- **Бренды-референсы:** Dior, Prada, Valentino, Ralph Lauren, Burberry, Hermès, Saint Laurent, Chanel (тяжёлый люкс — колониальная/милитари эстетика).
- **Формула:** *Сафари = летний милитари; Милитари = всесезонный сафари.*
- **Современное прочтение (2025–2026):** знаковые цвета + оверсайз/свободный крой + отдельные элементы (жилет Вассермана, карго).
- **Миксуется с:** классика, кэжуал, гранж, минимализм, драма; сафари — ещё с бохо, этно, кантри, романтизм.

### Деревенька / рустик (современный коммерческий)
- **Цвета:** природная гамма + наив (нежно-салатовый, розовый в принте), много «застиранного» молочного.
- **Принты:** мелкий цветочный (мильфлёр), коровий, мелкая неконтрастная клетка/виши.
- **Ткани:** простые натуральные (хлопок, ситец, сатин, бязь, лён, шерсть, овчина) — НЕ статусные (нет кашемира/шёлка/твида). Визуально не «дорого», про простоту.
- **Вещи:** размытый силуэт платьев/сарафанов, крупная вязка, стёганые жилеты, парки, шали, клетка-виши, каскадные юбки, плетёные сумки, деним-комбинезон.
- **Обувь:** резиновые сапоги, клоги, биркенштоки, угги, сабо.
- **Бренды-референсы:** Ganni, Paloma Wool, Doen; миддл — Around, Drugaya store.
- **Миксуется с:** спорт, классика (деловой), кантри, этно, гаучо, пижамный, винтаж.
- **Стоп:** драма в любом виде, бохо.

### Спорт-шик / Athleisure (натуральный + спорт)
- **Цвета:** нейтральные (чёрный, серый, белый, беж) + яркие акценты (электрик, неон-зелёный, огненно-красный).
- **Ткани:** мембранные/эластичные (нейлон, эластан, спандекс, полиэстер) + натуральные (хлопок, твид, шерсть, габардин, лён). Унисекс.
- **Вещи:** худи, зипка, свитшот, джоггеры, брюки с лампасами, легинсы/велосипедки, мягкий или структурный жакет, пальто мужского кроя, пижамные комплекты.
- **Обувь:** кроссовки, кеды, мягкие лоферы/мокасины, слипоны, сабо.
- **Аксессуары:** минималистичные, функциональные (рюкзак, кепка, очки, apple watch).
- **Бренд-референс:** Fear of God.
- **Принцип:** функциональность (карманы, молнии, завязки), всё удобное.

### Неофутуризм (future) — элементами
- **Цвета:** чёрный, белый + кислотные (фуксия, синий, неон-жёлтый, лайм).
- **Ткани:** высокотехнологичные, шуршащие/блестящие, искусственные волокна, LED/умные ткани.
- **Элементы:** рванина/необработанный край, разрезы, корсеты/портупеи, очки-маски космической формы, слим-комбинезоны, металлик, космические мотивы.
- **Бренды-референсы:** Coperni, Mugler, Monochrome, Sorelle.
- **Стоп:** в чистом виде не носибельно. Только элементы + классика/спорт/деним.

### Smart Casual / Casual chic / Чистый натуральный
- См. `sense-style-method.md` §4.4 и подстили №18, №25. Городской кэжуал, расслабленная элегантность, база + одна знаковая деталь.

## 4. Группы родства (для проверки сочетаемости)

- **Натуральная семья (свободно миксуется внутри):** чистый натуральный, бохо, гаучо, кантри, сафари, милитари, рустик, этно, английский загородный.
- **Классическая семья:** чистая классика, преппи, денди-гарсон, минимализм, авангард, парижский шик, дизайнерский, quiet luxury, smart casual, power woman.
- **Драматическая семья:** феминная драма, авангардная драма, готическая драма, деконструктивизм, гранж (частично), неофутуризм.
- **Романтическая семья:** чистый романтизм, наив, парижский романтизм, бельевой, леди-лайк.

**Несочетаемости (hard):**
- Авангард / интеллектуальный минимализм ✗ романтика, ✗ драма, ✗ тяжёлый люкс.
- Рустик ✗ драма, ✗ бохо.
- Неофутуризм в чистом виде ✗ повседневность.

## 5. Как этот файл используется

1. `look-generator.md` при сборке капсулы берёт из формулы клиентки её подстили → подтягивает отсюда цвета/ткани/вещи/правила микса → собирает образ, проверяя hard-rules раздела 1 и несочетаемости раздела 4.
2. `wardrobe-analyzer.md` при анализе гардероба определяет, к каким стилям относятся вещи клиентки, и предлагает миксы по правилам родства.
3. Бренды-референсы — только эстетический ориентир; в шопинг-листе подбираются аналоги под сегмент и бюджет клиентки (антипотребительский принцип: формула, а не куча вещей).
